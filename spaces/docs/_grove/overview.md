# grove documentation

This space **is** grove's documentation — written as a grove space, so the docs dog-food the tool
they describe. Browse it like any space: the tree on the left, a page in the middle, the terminal
below.

- [[guides/getting-started]] — install, run, and take the tour.
- **guides** — how to use grove day to day.
- **concepts** — the model and vocabulary.
- **internals** — how grove is built.
- **reference** — the command surface, as a queryable table.

Everything here is plain markdown with bold-label fields and `[[wikilinks]]`; edits go through the
same draft → commit → respin cycle as any other space.
